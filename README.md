## Deskripsi
Project ini bertujuan untuk memprediksi **status kelulusan siswa** berdasarkan nilai akademik dan data demografis menggunakan metode **Supervised Learning (Classification)**.

## Dataset
Dataset berasal dari Kaggle:
https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

File `students.csv` pada repository ini adalah **sample struktur data**.  
Silakan ganti dengan dataset asli dari Kaggle sebelum submit final.

## Algoritma yang Digunakan
- Logistic Regression
- Random Forest Classifier

## Tools
- Python
- Jupyter Notebook
- Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Kesimpulan
Model Random Forest memberikan performa lebih baik dibandingkan Logistic Regression karena mampu menangkap hubungan non-linear antar fitur.
